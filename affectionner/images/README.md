# images/

Drop your photos here and reference them in `index.html`.

## Naming convention

```
gallery-talk-wud-2024.jpg       ← World Usability Day talk
gallery-workshop-2024.jpg       ← workshop photo
gallery-event-rbi-2023.jpg      ← RBI conference
project-rbi-screenshot.png      ← work card screenshot
cert-google-ux.png              ← certificate badge
```

## Recommended sizes

| Use            | Size        | Format |
|----------------|-------------|--------|
| Gallery photos | 600×600px   | JPG    |
| Work card imgs | 800×500px   | JPG    |
| Cert badges    | 200×200px   | PNG    |

## How to use in index.html

```html
<img src="images/your-photo.jpg" alt="Descriptive caption"
     style="width:100%;height:100%;object-fit:cover;border-radius:6px;">
```
